# AutoFix CLI 🚀

Tired of mistyping terminal commands and googling stack traces? **AutoFix CLI** is here to help.
It catches your last terminal error and suggests the exact fix. 

## Features
* Detects common `git` typos (e.g., `git pus`).
* Detects missing Python modules.
* Interactive prompt: Just press `y` to execute the suggested fix.

## Usage
Just run the script:
```bash
python autofix.py
```
